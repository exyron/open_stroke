⚡ Gestos Multicapa

Combina pulsaciones de teclado con el botón central o derecho para multiplicar tus atajos. Dibuja en la pantalla y ejecuta comandos complejos al instante.

🛡️ Escudo de Titanio

Arquitectura inteligente basada en la API nativa de Windows (ctypes). El motor detecta automáticamente ventanas críticas y protege el ecosistema para evitar conflictos.

🎒 100% Portable

Ejecuta el entorno desde un pendrive. Crea un archivo portable.txt y OpenStroke guardará toda tu configuración localmente sin ensuciar el registro de Windows.
